# Module 03 - Introduction to Networking

This module covers core networking concepts required to understand how modern networks operate, how data is addressed and transmitted, and how secure communication is established.  
The focus is on networking models, addressing, protocols, secure communication mechanisms, and cryptographic foundations relevant for cybersecurity analysts, SOC roles, and blue-team operations.
